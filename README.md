# Gogh -- a WIP Wayland compositor written in C++ using Louvre
Yes, it's named after that one guy. This is mostly just a project for me to work on when I'm bored, not a main project. C++ is cool, I guess.

A lot of code is from the Louvre tutorials and Louvre Weston Clone, so you should go check out the [Louvre project](https://github.com/CuarzoSoftware/Louvre) if you want to build your own compositors in just a few hundred LoC.
